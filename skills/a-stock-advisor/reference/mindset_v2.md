# mindset v2 方法论摘要

> 完整文档见 `docs/mindset.md`。本文件是给 skill 和 subagent 看的速查版。

## 核心原则
1. **数据优先于消息**：先看资金在买什么，再看新闻有什么
2. **3 日趋势 > 单日波动**：单日是噪声
3. **大盘环境优先**：下跌趋势中所有策略都失效
4. **不追高**：板块/个股暴涨后等回调
5. **跟随资金**：不预测，看实际流向
6. **纪律重于预测**：止损永远比预测重要

## 七步法

### 第 0 步：大盘环境
- 上证 > MA20 且 MA5 > MA10 → 上升趋势（仓位 60-80%, 单票 20%, 最多 3 只）
- 在 MA20 ±3% 缠绕 → 震荡市（仓位 30-50%, 单票 15%, 最多 2 只）
- 上证 < MA20 且 MA5 < MA10 → 下跌趋势（仓位 10-20%, 单票 10%, 最多 1 只）

### 第 1-2 步：板块筛选
- 近 3 日行业涨幅排名
- 排除 3 日累计涨幅 > 15% 的过热板块
- 取排名 3-12 名（避开头部最热的，往往是接盘区）

### 第 3-4 步：个股筛选
- 排除 ST、新股（上市 < 1 年）
- 必须有分红记录
- 板块内 3 日累计涨幅在 30%-70% 分位
- ROE > 0（实盘可放宽）

### 第 5 步：技术面 5 维度打分（必须 >= 4 分）
1. 股价站上 MA20
2. MACD > 0 且 MACD_hist > 0
3. KDJ_J 在 20-80 之间
4. RSI(6) 在 30-70 之间
5. 收盘价 < BOLL 上轨

### 第 6 步：风险收益比 >= 2:1

### 第 7 步：操作建议
- 买入价: max(MA5, MA20) ~ 昨日收盘
- 止损: max(MA20, 买入价 * 0.93)
- 第一目标: min(近 30 日压力位, 买入价 * 1.08)
- 持有期: 2-4 周（短线）或 1-3 月（波段）

## 卖出规则
- 跌破 MA20 连续 2 天 → 全部止损
- 跌破买入价 -7% → 无条件止损
- 达到第一目标 → 卖 50%
- 达到第二目标 (+15%) → 全部卖出
- MACD 顶背离 + 放量滞涨 → 清仓
- 持有 > 20 个交易日未达预期 → 重审逻辑
