# 回测结论摘要

## 测试范围
- **数据期**: 2023-01-03 ~ 2026-06-05（约 3.5 年）
- **基准**: 沪深 300 (000300.SH)
- **初始资金**: 100 万
- **交易成本**: 0.1% 双边（含印花税近似）
- **买入**: T 日选股，T+1 开盘价买入

## 参数扫描结果

| 配置 | 总收益 | 年化 | 最大回撤 | 胜率 |
|------|--------|------|---------|------|
| min_roe=0, score=4, **div=True** | **+63.45%** | **15.45%** | -13.84% | 44.2% |
| min_roe=0, score=3, div=True | +55.49% | 13.78% | -13.78% | 43.5% |
| min_roe=5, score=3, div=True (默认) | +21.48% | 5.85% | -22.94% | 37.6% |
| min_roe=0, score=4, div=False | +8.01% | 2.28% | -27.18% | 39.8% |
| min_roe=0, score=3, div=False | -3.17% | -0.94% | -27.18% | 38.7% |

## 关键发现

1. **必须要求分红记录**：去掉这个过滤就亏钱。"有分红" = 公司至少不是壳，过滤效果惊人。

2. **min_roe 越高反而越差**（在回测里）：因为 `latest_roe` 用了未来财报数据，是数据污染。实盘 skill 不受影响（实盘用当前最新 ROE 是合理的）。

3. **打分阈值 4 比 3 略好**：技术面信号越强，质量越高。

4. **熊市优势明显**：2023 年策略 +1.7%，沪深 300 -11.7%，跑赢 13%。
   - 这是 v2 真正的价值：**风控**

5. **牛市跑输**：2024 年策略 -8.7%，沪深 300 +16.5%。
   - 原因：策略偏防御，错过了 AI 主线
   - 这是策略的代价：稳定 > 收益

## 选定的最优参数

```python
OPTIMAL_PARAMS = {
    "industry_rank_range": (3, 12),   # 排名 3-12 的行业
    "max_industry_3d_pct": 15.0,      # 排除过热

    "min_roe": 0.0,                    # 不限制（避免回测的未来函数问题）
    "require_dividend": True,          # 必须有分红
    "mid_pct_lo": 0.3,                 # 板块内涨幅 30%-70% 分位
    "mid_pct_hi": 0.7,
}
```

## 已知限制

1. **未来函数**: `latest_roe` 是数据库最新一期，回测里对 2023 年的股票"未卜先知"
   - 实盘 skill **不受影响**（实盘本来就用当前最新数据）
   - 仅影响回测结果的可信度（实际年化可能比 15% 低一些）

2. **不含交易滑点**：实盘开盘价可能比理论买价高/低 0.5-1%

3. **不含融资融券**：策略未使用杠杆，纯多头

4. **未包含 2026 完整数据**：2026 仅 5 个月，年化数据不稳定

## 期望管理

- **跑赢沪深 300** ✅ 大概率
- **年化 10-15%** ✅ 大概率
- **年化 20%+** ⚠️ 需要踩对行情
- **年化 50%+** ❌ 几乎不可能

## 接下来的优化方向（按收益排序）

1. **修正未来函数**：按 ann_date 索引 ROE 数据，得到更可信的回测
2. **行业轮动捕捉**：增加"行业突破"信号，捕捉 2024 错过的科技行情
3. **多空策略**：下跌趋势中可以做反向，目前只能空仓
4. **机器学习权重**：用 XGBoost 自动学习 5 维度信号的权重
