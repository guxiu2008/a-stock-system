# 中期波段模式 (swing)

## 目的

适合用户希望持有 **1~3 个月**，争取 **15~25%** 收益的中期波段，不追求 2~4 周短线快进快出。

## 与 short 模式区别

| 项目 | short | swing |
|------|-------|-------|
| 持有期 | 2~4 周 | 1~3 个月 |
| 目标 | +8% / +15% | +15% / +25% |
| 止损 | MA20 或 -7% | MA60 或 -10% |
| 行业依据 | 近 3 日强度 | 20 日 + 60 日趋势 |
| 技术打分 | >= 4 | >= 3 |
| 更重视 | 短期启动 | 中期趋势、行业强度、质量过滤 |

## swing 选股逻辑

1. 判断大盘环境，决定可操作数量
2. 计算行业中期强度：
   ```
   industry_strength = 20日收益 * 0.6 + 60日收益 * 0.4
   ```
3. 取行业强度排名第 3~15 名，避开最热头部
4. 个股过滤：
   - 非 ST / 非新股 / 非北交所
   - 有分红记录
   - 最新 ROE >= 1（过滤亏损或极弱公司）
   - 站上 MA60
   - 20日收益在 -5% ~ 35% 之间
   - 60日收益在 -15% ~ 80% 之间
   - 量能不过热（5日/20日量比 0.7~3.5）
5. 技术面至少 3/5 分
6. 输出 5 只候选：前 N 只为可操作，其余为观察池

## 风险提示

swing 模式不是长期价值投资，仍然是趋势策略。若买入逻辑失效或跌破 MA60 / -10%，必须重新评估或止损。
